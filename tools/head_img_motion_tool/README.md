# 会场头图 Lottie 本地工具

这个目录把 `jd-campaign-head-img-motion` 的核心流水线封装成本地网页工具。使用者不需要通过 AI 对话消耗 token，只要在浏览器里上传 2-4 个静态 JSON，就能生成循环 Lottie。

## 启动

双击 `start_tool.bat`。

也可以手动运行：

```powershell
C:\Users\honghaoxiang\.workbuddy\binaries\python\versions\3.13.12\python.exe server.py
```

默认地址：

```text
http://127.0.0.1:8787/
```

如果端口被占用，工具会自动使用后续可用端口。

## 使用

1. 选择 2-4 个静态 Lottie JSON，顺序就是画面顺序。
2. 工具会自动渲染静态画面，并识别每个画面的可选前景元素。
3. 可选：在静态画面上移动鼠标，高亮后单击元素，标记为“装饰元素”或“突出元素”。
4. 不标记任何元素也可以直接生成，此时只输出基础切换动效。
5. 点击“生成 Lottie”。
6. 在动效预览面板里输入文件名并保存最终 JSON。

## 动效选项

- 装饰元素：对应原 `--deco` 参数，元素会在画面停留阶段循环轻微浮动，适合花、蝴蝶、星星、氛围装饰。
- 突出元素：对应原 `--highlight` 参数，元素会在画面停留阶段做缩放强调，适合商品、核心利益点或主视觉焦点。

生成结果保存在：

```text
tools/head_img_motion_tool/runs/
```

## 说明

生成后静态标记区和动效预览区互不耦合，可以继续修改标记并重新生成。工具页外层的保存按钮会调用本地服务端弹出系统保存窗口；动效预览 iframe 只负责播放，不处理保存。
